\section{Supervised fine-tuning}

The Transformer section explains how a causal language model is trained. Supervised
fine-tuning (SFT) keeps that architecture and next-token machinery, but changes the data
distribution and, usually, which token targets contribute to the loss. The central
question is therefore not how to invent a new model objective, but how to convert desired
behavior into precise token-level supervision.

\section{Agentic reinforcement-learning traces}

The preceding GRPO derivation assumes that each training example is already a valid
autoregressive trajectory. Agent harnesses complicate that assumption by introducing
multiple model calls, tool responses, context replacement, subagents, and parallel
branches. These are data-construction problems that arise after the policy-update
algorithm has been defined.

\section{On-policy agentic traces}

The GRPO importance ratio is valid only when each optimized token was sampled by the
behavior policy recorded in its denominator. For an agent trajectory, the central question
is therefore whether a model action remains on-policy or crosses a boundary that requires
a new trajectory initialization.

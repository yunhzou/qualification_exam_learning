\begin{frame}[t]{Conclusion}
  \vspace{3mm}
  {\LARGE\textbf{Long-horizon agents need richer feedback than one terminal number.}}

  \vspace{4mm}
  \begin{enumerate}
    \setlength{\itemsep}{2mm}
    \item SFT, RL, OPD, OPSD, and MOPD solve different supervision problems.
    \item As trajectories grow, feedback must move from outcomes to responsible traces and corrected token distributions.
    \item Source-grounded curricula and verified model updates can turn post-training into continual domain adaptation.
  \end{enumerate}

  \vspace{2mm}
  \begin{center}
    {\Large\color{DeckTeal}\textbf{Harnesses define the scientific process;\\post-training develops the intelligence inside it.}}
  \end{center}
\end{frame}
